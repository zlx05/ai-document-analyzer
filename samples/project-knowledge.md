# AI 知识库 Agent 平台项目资料

## 项目定位

本项目是一个面向个人开发者和小团队的 AI 项目管理平台。用户可以上传项目资料，系统会将资料解析、切片并存入私有知识库。Agent 会基于项目资料和任务数据，自动拆解任务、总结项目进度，并记录每一次工具调用过程。

## 核心目标

1. 让用户可以上传自己的项目资料，形成项目级私有知识库。
2. 让 Agent 不只是回答问题，而是可以调用后端工具查询任务、创建任务和总结项目状态。
3. 让前端展示 Agent 的执行过程，包括检索知识库、查询任务、创建任务和生成总结。
4. 让项目具备实习面试展示价值，覆盖前端、后端、数据库、RAG、Agent 和测试。

## 主要功能

- 用户注册和登录
- JWT 权限控制
- 项目创建和项目列表
- 任务创建、状态更新和任务统计
- 文件上传和文本解析
- 文档切片入库
- RAG 知识检索
- Agent 工具调用
- Agent 执行记录和工具调用轨迹
- 接口测试和接口文档

## 数据库设计

系统使用 MySQL 存储业务数据。核心表包括：

- users：用户信息
- projects：项目数据
- tasks：项目任务
- documents：上传文件记录
- document_chunks：文档切片
- agent_runs：Agent 运行记录
- agent_tool_calls：工具调用记录

用户和项目是一对多关系，项目和任务是一对多关系，项目和文档是一对多关系，文档和切片是一对多关系，Agent 运行和工具调用是一对多关系。

## Agent 工具

Agent 可以调用以下工具：

- retrieve_knowledge：检索当前项目知识库
- list_tasks：查询当前项目任务
- create_tasks：根据资料自动创建任务
- summarize_project：统计项目任务状态并生成总结

## 项目亮点

本项目不是一个普通聊天机器人，而是一个能够读写业务系统的 Agent。Agent 会先读取项目私有知识库，再结合数据库里的任务状态进行操作。每次执行都会保存工具调用过程，因此具备可追踪、可调试和可审计的特点。
